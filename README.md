# Instructor LOs

A React app built with [Vite](https://vite.dev/) and [Tailwind CSS](https://tailwindcss.com/).

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Running code from ChatGPT

Paste a React component into `src/App.jsx` (replace the existing `App` component), and the dev server will hot-reload automatically. Tailwind classes work out of the box.

If the component needs an extra library (e.g. `recharts`, `lucide-react`), install it first:

```bash
npm install <package-name>
```
